Requirements and Installation
=============================

Requirements
------------

Tested with:

* Python: 2.7, 3.3, 3.4, 3.5.
* pyparsing: 2.2.0+
* six: 1.10.0+


Installation
------------

Install with ``pip``:

.. code-block:: console

   $ pip install mysqlparse

Or add it to your project's ``requirements.txt``.
